## Run Locally

Clone the EdgarHdzDev

```bash
  git clone https://github.com/EdgarHdzHdz17/EdgarHdzDev.git
```

Go to the project directory

```bash
  cd EdgarHdzDev
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
